# Technology:

## Client:

- Reactjs
- Redux(State Management)

## Server:

- Nodejs
- ExpressJs
- Mongodb(atlas)

## Config directory req a file default.json which should contain:

- _mongoUri_
- _mongoURI_
- _mongoLocal_
- _jwtsecret_
- _githubClientId_
- _githubClientSecret_
